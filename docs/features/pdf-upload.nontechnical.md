## PDF Upload (Non‑Technical Overview)

### What this feature does
- Lets you **choose a PDF file from your computer**.
- Shows that the file was successfully loaded (by displaying its name).
- Prepares the document so you can **start placing variables** on it.

### What happens behind the scenes (in simple terms)
- The app reads your PDF into memory so it can show a preview.
- Any variables you added for a previous PDF are **cleared**, so you always work with a clean template for the current file.

### What this feature does *not* do
- It does **not** edit or change your original PDF file.
- It does **not** upload the file to a server in this project; everything stays in your browser session.
