## Template Export to JSON (Non‑Technical Overview)

### What this feature does
- Turns your current PDF template (all placed fields and their settings) into a **downloadable JSON file**.
- The JSON file can be used by **other tools or services** to know:
  - Which fields exist.
  - Where they are on each page.
  - What type of data they expect.

### How you use it
- After placing and configuring your fields:
  - Click **“Export JSON”** in the right‑hand inspector.
  - A `.json` file is downloaded, usually named after your PDF.

### Why it matters
- JSON is a **machine‑readable format**, which makes it easy for developers or backend systems to:
  - Fill in the PDF using real data.
  - Store or version your template definitions.
- This is the bridge between your **visual template design** and **automated PDF generation**.
