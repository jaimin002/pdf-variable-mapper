## Variable Placement (Non‑Technical Overview)

### What this feature does
- Lets you **drag variable types** (like Text, Number, Date, Email, Signature) from the left panel onto the PDF preview.
- Drops become **fields on the page**, representing where information will be filled in later.

### How it feels to use
- You pick a variable type from the left.
- Drag it over the page and release where you want the field to appear.
- A small labeled box appears at that position so you can see your template layout.

### Why it matters
- This is how you **design a fillable template** on top of a static PDF.
- It removes the need to hard-code coordinates or modify the original PDF — you just place fields visually.
